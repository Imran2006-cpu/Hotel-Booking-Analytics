# 🏨 Hotel Booking Analytics

An **End-to-End Data Analytics Project** that explores hotel booking patterns, customer behavior, revenue performance, and cancellation trends using **Python, SQL, Machine Learning, and Power BI**.

---

## 📌 Project Overview

This project demonstrates a complete data analytics workflow, starting from raw hotel booking data and ending with interactive business dashboards.

The workflow includes:

- 🧹 Data Cleaning & Preprocessing
- 📊 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning
- 🗄️ SQL Business Analysis
- 📈 Interactive Power BI Dashboard

---

# 🎯 Project Objectives

- Analyze hotel booking trends.
- Understand customer booking behavior.
- Measure revenue performance.
- Identify cancellation patterns.
- Build interactive dashboards for business users.
- Apply machine learning techniques for predictive analysis.

# 📈 Key Business Insights

- 🏨 City Hotel generated higher revenue than Resort Hotel.
- 🌐 Online Travel Agencies contributed the highest number of bookings.
- ❌ Longer lead-time bookings experienced higher cancellation rates.
- 👥 Repeat guests represented a relatively small percentage of total customers.
- 📅 Booking demand varied significantly across different months.
